# Vibe: Dark Tech / Devtool

**Vibe:** VSCode, GitHub, Linear dark, Vercel. Functional, technical.

| Aspek | Karakteristik |
|-------|--------------|
| **Layout** | Grid regular. Sidebar + content. Split panels. Data-dense |
| **Color** | Dark bg (navy/neutral, jangan pure black). 1 accent neon (cyan, green, purple) |
| **Font** | Mono untuk code. Sans untuk UI. JetBrains Mono, SF Mono |
| **Corner radius** | 6-8px. Subtle. Jangan pill |
| **Border** | `border-white/[0.06-0.1]`. Near-invisible |
| **Motion** | Minimal. Fade. Hover state via border/warna |
| **Cards** | Border-only. Atau bg elevated. No shadow |
| **Typography** | Smaller scale. Dense. Mono untuk detail |
| **Reference** | github.com, vercel.com dark, linear.app, supabase.com |

**Dials:** VARIANCE: 4-5 | MOTION: 2-3 | DENSITY: 5-7
