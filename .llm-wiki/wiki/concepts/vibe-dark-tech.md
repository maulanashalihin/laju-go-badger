# Vibe: Dark Tech / Devtool

**Vibe:** VSCode, GitHub, Linear dark, Vercel. Functional, technical.

| Aspect | Characteristics |
|-------|--------------|
| **Layout** | Regular grid. Sidebar + content. Split panels. Data-dense |
| **Color** | Dark bg (navy/neutral, not pure black). 1 neon accent (cyan, green, purple) |
| **Font** | Mono for code. Sans for UI. JetBrains Mono, SF Mono |
| **Corner radius** | 6-8px. Subtle. No pill |
| **Border** | `border-white/[0.06-0.1]`. Near-invisible |
| **Motion** | Minimal. Fade. Hover state via border/color |
| **Cards** | Border-only. Or elevated bg. No shadow |
| **Typography** | Smaller scale. Dense. Mono for details |
| **Reference** | github.com, vercel.com dark, linear.app, supabase.com |

**Dials:** VARIANCE: 4-5 | MOTION: 2-3 | DENSITY: 5-7
