# Vibe: Playful / Agency / Experimental

**Vibe:** Awwwards winners, Stripe press, Vercel, Dribbble featured. Individual, avant-garde.

| Aspek | Karakteristik |
|-------|--------------|
| **Layout** | Grid tidak konvensional. Overlap. Scroll pinning. Split. Asimetris ekstrim |
| **Color** | Bold. Saturated accent. Bisa 2 accent. Gradient berani. Dark mode preferred |
| **Font** | Display sans ekstrim. Variable font weight extreme. Bisa mono di body |
| **Corner radius** | Variasi. Ada sharp, ada pill. Intentional |
| **Border** | Heavy `border-2` atau none. Tidak ada tengah-tengah |
| **Motion** | Cinematic. GSAP ScrollTrigger, physics, magnetic hover, parallax |
| **Typography** | Oversize font scale. Tracking extreme. Letter-spacing play |
| **Reference** | stripe.com, vercel.com, codrops.com, awwwards.com |

**Dials:** VARIANCE: 9-10 | MOTION: 8-10 | DENSITY: 3-4
