# SQLite

**SQLite** is the database engine for Laju Go, using the pure-Go `modernc.org/sqlite` driver (CGO-free) initially, later migrated to `github.com/mattn/go-sqlite3` (CGO-based).

## Role in Laju Go

- Database-backed sessions via SQLite table
- Migrations via Goose (run automatically on startup)
- Queries generated by sqlc from `queries/*.sql`
- Schema source in `migrations/` directory

## Conventions

- One table per migration file
- Each migration file must have `-- +goose Up` and `-- +goose Down` sections
