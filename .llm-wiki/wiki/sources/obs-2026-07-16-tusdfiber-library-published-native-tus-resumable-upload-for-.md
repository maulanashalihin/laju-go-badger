# ⭐ Observation: tusdfiber library published — native TUS resumable upload for Go Fiber
User published tusdfiber (github.com/maulanashalihin/tusdfiber), a native Go Fiber implementation of TUS resumable upload protocol (v1 + IETF v2 draft). Features: no adaptor.HTTPHandler(), streaming body, CORS, concatenation, deferred length, notifications, callbacks, hooks system (file/HTTP/gRPC via tusd), Prometheus metrics, locking. 51 unit tests + E2E. Relevant to Laju Go since it uses Go Fiber and may need resumable upload for large files in the future.
